# M6 Projekt

## Om projektet

Dette projekt er en statisk prototype hjemmeside, der præsenterer opskrifter og måltidsidéer til morgenmad, frokost, aftensmad og snacks. Siden er bygget med HTML, CSS og JavaScript, og den indeholder flere opskriftssider samt et samlet layout i `index.html`.

## Indhold i projektet

- `index.html` — hovedsiden med navigation til de forskellige måltidskategorier.
- `morgenmad.html` — side med morgenmadsopskrifter.
- `frokost.html` — side med frokostidéer.
- `aftensmad.html` — side med aftensmadsopsætning.
- `snack.html` — side med snackforslag.
- `bolognese_opskrift.html`, `havregrynskugler_opskrift.html`, `hoensesalat_opskrift.html`, `vafler_opskrift.html` — individuelle opskriftssider.
- `assets/css/styles.css` — styling for hele websitet.
- `assets/js/app.js` — JavaScript til interaktivitet og funktionalitet.
- `assets/img/` — mappe til billeder brugt på siden.

## Opsætning

1. Klon eller kopier projektmappen til din computer.
2. Åbn mappen i din foretrukne kodeeditor.
3. Åbn `index.html` i en webbrowser for at se websitet.

> Der er ingen server-installation nødvendig, da det er en ren statisk hjemmeside.

## Sådan kører du projektet lokalt

- Åbn `index.html` direkte i browseren.
- Alternativt kan du bruge en lokal HTTP-server, f.eks. via Visual Studio Code med Live Server-udvidelsen.
